# 画面要件定義 v1: 設定（C-02）

**作成日**: 2026-02-22

---

## 1. 機能要件

| # | 機能 | 説明 |
|---|------|------|
| F01 | 難易度デフォルト設定 | 練習設定画面のデフォルト難易度（初級/中級/上級）をセレクターで設定する |
| F02 | 制限時間ON/OFF | 練習問題に制限時間を設けるかをトグルで設定する（デフォルト: OFF） |
| F03 | 分量クイズ有効化 | 分量クイズ問題を練習セッションに含めるかをトグルで設定する（デフォルト: OFF） |
| F04 | ハプティクスフィードバック | ドラッグ&ドロップ時のハプティクスをON/OFFする |
| F05 | データリセット | 全ての学習データ（練習記録・振り返り・間違い問題リスト・習得ステータス）を初期化する |
| F06 | アプリバージョン表示 | 現在のアプリバージョンを表示する |

## 2. 非機能要件

| # | 項目 | 要件 |
|---|------|------|
| NF01 | 設定保存 | 設定変更を即座にSQLite（またはAsyncStorage）に保存し、次回起動時も維持すること |
| NF02 | データリセット速度 | リセット処理を2秒以内に完了すること |

## 3. 表示条件・バリデーション

| 条件 | 処理 |
|------|------|
| データリセットボタン押下 | 「本当にすべてのデータをリセットしますか？この操作は取り消せません。」確認ダイアログを表示 |
| ダイアログでキャンセル | リセット処理を行わずダイアログを閉じる |
| ダイアログで確認 | USER_PROGRESS / PRACTICE_SESSION / PRACTICE_RESULT / REVIEW_NOTE / WRONG_ANSWER の全レコードを削除する |

## 4. エラーケース

| エラー | 対応 |
|--------|------|
| 設定保存失敗 | 「設定の保存に失敗しました」トースト通知 |
| データリセット失敗 | 「リセットに失敗しました。アプリを再起動して再試行してください」表示 |
