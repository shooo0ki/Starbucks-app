## ペルソナ1: 田中 さくら

> 「レシピが多すぎて全然覚えられない。シフト前に少しでも予習したい。」

### 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | 田中 さくら（Tanaka Sakura） |
| 年齢 | 19歳 |
| 役職 | アルバイト（入社1ヶ月） |
| 居住地 | 東京都 大学近隣 |
| デバイス | iPhone 15（主にスマホで情報収集） |

### 背景・経験

大学1年生で、スターバックスが初めてのアルバイト経験。飲食店のアルバイト自体も初めて。SNSでスタバ好きが高じてアルバイトに応募した。現在はドリンクの数の多さと、サイズや分量の組み合わせに圧倒されている状態。先輩スタッフに質問するのが恥ずかしく、シフト終わりに1人でメモを取り直すことが多い。スマートフォンは1日5時間以上使用し、SNSやYouTubeが主な利用コンテンツ。

### ゴール（このアプリで達成したいこと）

- 46品のドリンクレシピをすべて覚えたい
- シフト前に「今日出そうなドリンク」を素早く確認したい
- 自分がどのドリンクを苦手としているかを把握したい
- 先輩に質問しなくても、1人で自習できる環境が欲しい

### 課題・フラストレーション

- ドリンクの数が多すぎて、何から覚えればいいかわからない
- 手順の細かいパラメータ（スチーム秒数・シロップポンプ数等）が頭に入らない
- カスタムオーダーに対応できる自信がない
- 紙のメモは散逸しやすく、すぐに確認できない

### 行動特性

- スキマ時間（通学電車の中など）に短時間で学習しようとする
- 「やること」が具体的で小さいと取り組みやすい
- 視覚的にわかりやすいUI を好む（カード形式・バッジ等）
- ゲーム感覚で進捗が見えると継続しやすい

### 利用シーン

| シーン | 内容 |
|--------|------|
| 通学中（電車内） | レシピ一覧を眺めて、まだ覚えていないドリンクを確認する |
| シフト前（自宅） | 練習モードで5〜10分間ドリル練習をして予習する |
| シフト後（帰宅後） | 今日間違えた内容を振り返り記録に書く |

## ペルソナ2: 鈴木 けんた

> 「ホットはなんとかなるけど、フラペチーノの分量がまだ怪しい。集中的に練習したい。」

### 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | 鈴木 けんた（Suzuki Kenta） |
| 年齢 | 21歳 |
| 役職 | アルバイト（入社3ヶ月） |
| 居住地 | 神奈川県 |
| デバイス | Android スマートフォン（Galaxy S24） |

### 背景・経験

大学3年生で、カフェでのアルバイト経験は今回が初めて。入社から3ヶ月経ち、ホット系・アイス系のドリンクはおおよそ対応できるようになってきたが、フラペチーノ系やカスタムの多い注文に自信がない。勉強が忙しい時期には練習時間が取れないことが多く、習得にムラが生じている。Android ユーザーで、Google カレンダーや Notion でタスク管理をしているなど、デジタルツール活用に慣れている。

### ゴール（このアプリで達成したいこと）

- 苦手なフラペチーノ系・カスタム対応の習得精度を上げたい
- 間違えやすいドリンクを集中的に練習したい
- 週単位で自分の進捗を見て、学習ペースを管理したい
- 期間限定メニューが変わったとき、素早く覚えたい

### 課題・フラストレーション

- 勉強の忙しさでアプリの起動が途切れると、せっかく覚えかけたことを忘れる
- フラペチーノの分量（シロップ・パウダー量）が微妙にあやふや
- 期間限定メニューはメモに書いたが見づらく、練習に使えていない
- 自分が「何を知っていて、何を知らないか」が整理できていない

### 行動特性

- 目標を数値化して管理するのが好き（KPI 志向）
- 弱点を特定したら集中的に補う「弱点克服型」の学習スタイル
- ある程度まとまった時間（20〜30分）で集中して練習するタイプ
- 期間限定メニューへの関心が高く、自分でレシピを調べてメモする習慣がある

### 利用シーン

| シーン | 内容 |
|--------|------|
| 休日の夕方（自宅） | 間違い問題リストを確認し、弱点ドリンクに絞った再練習を20〜30分行う |
| シフト前夜 | 新しく導入された期間限定メニューをユーザー登録し、翌日のシフトに備える |
| 週末の振り返り | 今週の練習セッション結果を確認して、来週の目標（習得したいドリンク）を決める |

## ペルソナ3: 山田 あおい

> 「自分はもう大体覚えてきたけど、後輩に教えるためにも知識を整理したい。」

### 基本情報

| 項目 | 内容 |
|------|------|
| 名前 | 山田 あおい（Yamada Aoi） |
| 年齢 | 22歳 |
| 役職 | アルバイト（入社5ヶ月・シフトリーダー候補） |
| 居住地 | 大阪府 |
| デバイス | iPhone 14 |

### 背景・経験

大学4年生。スターバックスで5ヶ月のバイト経験を積み、ホット・アイス・フラペチーノを含むほぼすべてのメニューに対応できる。近々シフトリーダーを打診されており、後輩の指導にあたる機会が増えつつある。自分の知識の「抜け」をゼロにしたいという向上心が高い。新しい期間限定メニューはいち早く覚え、後輩へのレシピ共有にも役立てたいと考えている。時間の使い方が効率的で、スキマ時間を最大限に活用する。

### ゴール（このアプリで達成したいこと）

- 自分の知識の抜けを最終確認し、100% の習得率を達成したい
- 期間限定メニューをアプリに登録して、後輩に説明するための参照資料にしたい
- マルチタスク（10オーダー並び替え）の判断力を磨きたい
- 正解率・習得率などの統計データを見て、自分の成長を確認したい

### 課題・フラストレーション

- 細かいカスタム（非乳性ミルク使用時の手順差異など）を正確に説明できるか不安
- 期間限定メニューが口頭・メモでしか共有されず、正確な情報を保持しにくい
- 現状の学習ツールが存在しないため、自分で教材を作る手間がかかっている

### 行動特性

- 網羅性を重視し、「抜け」をなくすことにこだわる
- 他者（後輩）への説明・指導に活かせるツールに価値を感じる
- ユーザー登録機能など、カスタマイズ可能な機能を積極的に活用する
- アプリの習熟度が高く、細かい機能まで活用しようとする

### 利用シーン

| シーン | 内容 |
|--------|------|
| 休憩時間（店舗内） | 期間限定メニューをアプリに登録し、練習対象に追加する |
| 自宅（夜） | 全ドリンクの習得率ダッシュボードを確認し、未習得ドリンクを洗い出す |
| シフト前（朝） | 10オーダー並び替え練習を行い、マルチタスク判断力を磨く |
